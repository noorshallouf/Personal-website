مجالات التركيز

هذه بعض التطبيقات والمواقع التي قمت ببنائها مؤخرًا.

تصميم UI/UX

تصميم واجهات تركّز على المستخدم تجمع بين الجمال وسهولة الاستخدام باستخدام أدوات مثل Figma للنمذجة والتعديل.

تطوير الواجهة الأمامية

بناء واجهات تفاعلية واستجابية باستخدام HTML وCSS وأطر عمل JavaScript بشكل جذاب وسهل الوصول.

Back-end بلارافل

تطوير تطبيقات ويب آمنة وقابلة للتوسع باستخدام Laravel، مع بناء واجهات API، إدارة قواعد البيانات، وتفعيل أنظمة التوثيق.

استعراض مشاريعي

واجهة تطبيق حياة عصرية بفلتر

تطبيق Flutter (واجهة فقط)

صممت واجهة أنيقة وتفاعلية لتطبيق يخص نمط حياة النساء الشابات، مع التركيز على المكوّنات النظيفة والحركات السلسة والتنقل البسيط.

Back-end

Laravel

نظام خلفي متكامل لمتجر إلكتروني تم تطويره باستخدام Laravel. يدعم إدارة المنتجات والطلبات والمستخدمين، ويحتوي على لوحة تحكم ديناميكية باستخدام Filament لتسهيل تجربة إدارة المتجر.

نظام إدارة المخزون

تطبيق سطح مكتب Java + Hibernate

تطبيق مكتبي باستخدام Java SWT وHibernate لإدارة المخزون والمبيعات لمحل بيع بالتجزئة. يساعد على تنظيم العمليات وتسجيل المبيعات بدقة.
